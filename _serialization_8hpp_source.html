9// A base class/struct used for denoting that a class can be scanned.
10// Reflectable values are int, uint, bool, float, string, Reflectable, and std::vectors of the previous types.
11// Reflectables are interepreted as headers of the provided name.
12// Circular references are not handled and will crash.
13// Marking a field as reflectable without properly initalizing it will crash upon attempted deserialization.
14// Ensure that the default CTOR initalizes all fields.
15struct Reflectable {
16 virtual ~Reflectable() = default;
17 virtual std::string File() { return std::string(); };
18};
